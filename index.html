Learn Mongo database

Basic Command For Mongo DB

  • for start the mongo db
    • Syntax: mongo
  • For show database
    • Syntax: show dbs
  • For Create Database or Select Exitsting Database
    • Syntax: use 'dbName'
    Example: use learnmongodb
  • for create Collection(Table)
    • Syntax: db.collectionName.insertOne({field:value})
    Example: db.mydata.insertOne({name:'React Js', type:'Front End', Videos:80, active:true})
  • show active database name
    • Syntax: db
  • For show the collections(tables) inside database
    • Syntax: show collections
  • for show documents inside the collection(table)
    • Syntax: db.collectionName.find()
    Example1: db.mydata.find()
    Example2: db.mydata.find().pretty();
  • for exit from mongo db
    • Syntax: quit()
    Syntax: ctrl+c

CRUD Operation of Mongo DB

First Start Mongo DB
Syntax: mongo

  1. Create

    • Create Database
      • Syntax: use databaseName
      Example: use learnmongodb
    • Insert Collection(table) in Database with Documents
      • Syntax 1: db.collaction.insertOne()
      for insert One documents

      Example: db.users.insertOne({name:'Rabi Kr Yadav', age:26, status:true})

      Syntax 2: db.collaction.insertMany()
      for insert Multiple documents using Array of Object

      Example: db.users.insertMany([{name:'Krishan', age:25, status:false},{name:'Rakesh', age:24, status:false}])

  2. Read

    3. Syntax: db.collection.find(query, projection)
    • Show Database list
      • Syntax: show dbs

    • Select Database
      • Syntax: use databaseName Example: use users

    • Show Selected Database
      • Syntax: db

    • Show Collections
      • Syntax: show collections)

    • Find all Documents(data) inside collection(table)
      • Syntax: db.collection.find()

    • Find all Documents in Pretty Format
      • Syntax: db.collection.find().pretty()

    • Find custom document with query
      • Syntax: db.collection.find(query)
      Example: db.collection.find({name:'Rabi Kr Yadav'})

    • Find custom document with custom query(feild)
      • Syntax: db.collection.find(query,projection)
      Example 1: db.collection.find({name:'Rabi Kr Yadav'},{name:1})

      Example 2: db.collection.find({name:'Rabi Kr Yadav'},{_id:0, name:1})
      Example 3: db.collection.find({name:'Rabi Kr Yadav'},{_id:0, name:1, age:1})
      Note: where 0 for false (hide) or 1 for true (show) the feilds
    • show only limited field from Many Matching Documents
      • Syntax 1: db.users.find(query).limit(value)
      Example: db.users.find({status:true}).limit(2)

      Syntax 2: db.users.findOne(query)
      Example: db.users.findOne({status:true})

    • show only limited field from Many Matching Documents with skip first document
      • Syntax: db.users.find(query).pretty().limit(value).skip(value)
      Example: db.users.find({status:true}).pretty().limit(2).skip(1)

  3. Update

    4. Syntax for UpdateOne: db.collectionName.updateOne(filter, update)
    Example: db.users.updateOne({name:'Krishan'}, {$set: {name:'Krishna'}})
    Syntax for UpdateMany: db.collectionName.update(filter, update)
    Example: db.users.updateMany({name:'Krishna'}, {$set: {status:false}})
    Note: The $set operator replacex the value of a field with the specified value

  4. Delete

    5. Syntax: db.collectionName.deleteMany(deletation_Criteria)
    Example: db.users.deleteMany({age:20})
    Syntax: db.collectionName.remove(deletation_Criteria)
    Example: db.users.remove({age:28})
    For Delete all documents
    Syntax: db.collectionName.deleteMany({})
    Example: db.users.deleteMany({})
